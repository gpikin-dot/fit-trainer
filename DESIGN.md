# FitTrainer — Design System

Источник истины по визуальному языку приложения. Эталон-экран: `04_Tech/fittrainer-redesign-proposal-9.html` (Quiet minimal, вариант 9).

## Концепция

«Бумага и тушь» — тихий минимализм. Инструмент, который не мешает: данные первичны, интерфейс отступает. Подходит для зала (читаемость с расстояния, одна рука).

- **Почти монохром.** Цвет — носитель смысла, не украшение. Единственный цветовой сигнал — зелёный для «выполнено».
- **Цифра-герой.** Повторы / вес / время — самый крупный и тихий элемент (тонкое начертание, большой кегль). Контраст строится на **весе шрифта**, не на цвете.
- **Воздух и волосяные линии.** Разделение — тонкими линиями и пространством, а не заливками и тенями.
- **Текущий шаг — единственный выделенный блок** на экране.

## Шрифт

**Onest** (Google Fonts) — гротеск с полной кириллицей, нейтральный с лёгким характером. Один семейство, контраст через вес.

- Гигантские цифры / hero: `300` (light)
- Заголовки экранов: `600`
- Текст, кнопки, подписи: `500–600`
- Акцент-эмфаза: `700`
- Лейблы-капс: `600`, `letter-spacing .18–.30em`, `uppercase`, цвет `--ink-faint`

Никаких засечек. Никаких Inter/Roboto/системных.

## Палитра (токены)

| Токен | Hex | Назначение |
|---|---|---|
| `--paper` | `#fbfaf7` | фон страницы |
| `--paper-2` | `#f2f1ec` | подложка активного блока, мягкие зоны |
| `--ink` | `#1c1b18` | основной текст, акцент, primary-кнопка, прогресс |
| `--ink-dim` | `#76736a` | вторичный текст |
| `--ink-faint` | `#a7a499` | лейблы, плейсхолдеры, неактивное |
| `--hair` | `#d8d5cc` | волосяные линии, неактивные бордеры |
| `--line-strong` | `#1c1b18` | сильные разделители (под заголовком, навигация) |
| `--pos` | `#3f7d4f` | единственный цвет — «выполнено» |
| `--pos-soft` | `#eef3ec` | фон-намёк «выполнено» |

Акцент = `--ink`. Синего/градиентов/неона нет.

## Форма

- Радиусы плоские: блоки `6–10px`, инпуты/кнопки `4–8px`, чипы — круг.
- Тени почти не используются (только активный шаг — лёгкая, опционально).
- Бордеры — `1px` `--hair`; сильные акценты — `2px` `--ink`.
- Тап-таргеты ≥ 44px.

## Ключевые паттерны

- **Таймер отдыха** — горизонтальный, компактный: сверху мелкий капс-лейбл + текстовые кнопки (`Пропустить · +30 сек`), ниже крупное время (`78px`, вес 300) и под ним жирная «убегающая» полоса остатка (бегущий блик). Без круга.
- **Подходы** — таблица: `#` · крупная цифра повторов · крупная цифра веса · круглая отметка. Заголовок колонок капсом; единицы (кг/повт) НЕ дублируем у цифр.
- **Текущий подход** — подложка `--paper-2`, нижняя граница `2px --ink`, цифры крупнее, отметка с пульсом.
- **Степпер упражнений** — равные сегменты с нижней линией: текущий `2px --ink` + bold, выполненный — `--pos`.
- **Шапка** — бренд капсом `letter-spacing .36em`, заголовок экрана `600`, граница-разделитель `1px --ink`.
- **Прогресс** — волосяная линия `1px --hair` с заполнением `3px --ink`.
- **Появление** — лёгкий `translateY` + `opacity`, стаггер 0 / .1s / .16s.

## Реализация

Перекраска идёт через `src/tokens.css` (централизованные переменные). Существующие имена токенов (`--blue-*`, `--slate-*`, `--green-*`, `--font`) переотображены на новую палитру → весь app получает язык без правки каждого экрана. Точечно дорабатываются только бесшаблонные места ядра (таймер, размеры цифр).

Раскатка: ядро (тренировка/сессия/таймер) → остальные экраны. Только `develop` (staging) до явного ОК на production.
